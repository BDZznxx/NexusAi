## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm atau yarn

### Installation

```bash
# Clone repository
git clone https://github.com/nexus-ai-platform/nexus-ai.git
cd nexus-ai

# Install dependencies
npm install

# Copy environment file
cp .env.example .env

# Start development server
npm run dev
